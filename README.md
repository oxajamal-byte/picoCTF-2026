# picoCTF-2026

My first CTF ever. Competed solo in picoCTF 2026 run by Carnegie Mellon University. The competition ran March 7 through 17 and had over 90,000 teams globally and 14,000 in the US high school division. Teams can have up to five members. I did it alone.Finished top 100 in the US and 530 globally. Solved around 65 challenges across all six categories.This repo has writeups for the challenges I found most interesting or educational. Not every challenge is here because some were straightforward enough that a writeup wouldn't add much. I focused on the ones where I actually learned something worth documenting.Categories covered:

General Skills
Forensics
Cryptography
Web Exploitation
Binary Exploitation
Blockchain
Tools I used throughout the competition: Python, pwntools, GDB, Ghidra, Burp Suite, Wireshark, foremost, binwalk, SageMath, CyberChef, netcat, and a lot of terminal commands on Kali Linux.
